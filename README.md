Project Summary:  Finly – AI-Powered Loan Recovery Ecosystem

For a detailed breakdown of the data analysis and predictive modeling, please see the full project report.

Addressing the challenges of an NBFC's inefficient, one-size-fits-all loan collection process, this project introduces a closed-loop, AI-powered ecosystem designed to shift collections from a reactive chore into a proactive, customer-centric operation.

Core Pillars of the Solution
1. High-Accuracy Prediction
At its core is a highly optimized Random Forest machine learning pipeline. By aggressively stripping out 37% of structural data "noise," the model predicts payment defaults with an exceptional 94.0% accuracy, successfully isolating high-risk accounts for real-time inference without computational bloat.

2. Innovative Customer Segmentation
The key innovation is a behavioral feature engineering process that moves beyond simple binary predictions. It calculates a custom Customer_Friction_Index and segments users into data-driven Borrower Personas (e.g., "Tough Youth," "Cooperative Adult"). This contextualizes the why behind a default and solves the "Positive Sentiment Paradox."

3. Intelligent Automation
The predictive models and personas feed directly into Finly, a state-aware generative AI assistant powered by Gemini 2.5 Flash. Rather than reading a static script, the chatbot:

Dynamically adapts its negotiation tone based on the user's specific persona.

Offers real-time restructuring, such as calculating split-payment calendar dates on the fly.

Operates within strict banking guardrails, including an "Anti-Loop Guard" and a "User Initiative Override" to seamlessly pivot strategies and secure recovery.
